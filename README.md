# AI Media Lab

Minimal scaffold for the AI Media Lab project. This repository contains a Next.js frontend, a FastAPI backend, and microservices for ComfyUI-based model orchestration. Deployment target (Option A): Vercel (frontend), RunPod GPUs (ComfyUI), Cloudflare R2 (storage), Supabase (metadata).

See `.github/instructions/deployment.instructuions.md` and `docs/` for runbooks and licenses.
