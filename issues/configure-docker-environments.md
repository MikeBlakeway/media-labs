## User Story
As a developer, I want to configure Docker Compose environments for both CPU-only and GPU-enabled development, so that I can reliably run and test all Media Lab services on my local machine or a GPU host.

## Acceptance Criteria
- Docker Compose files exist for both CPU-only and GPU-enabled setups.
- Documentation covers how to start the environment in either mode, detailing required images and configuration steps.
- All core services (web, API, worker services) start successfully using Docker Compose.
- Troubleshooting steps are included for common Docker issues.
